##如何安裝和運行
____
### 克隆存儲庫
 
```sh
$ cmd
$ git clone https://github.com/BEPb/System_monitor
```
 
### 安裝必要的包（安裝依賴）
```sh
$    -r requirements.txt
```

＃＃ 用法
在終端中，切換到程序目錄：
```
$ cd System_monitor
```
並輸入：
```
python sysmonitor.py
```
要啟動 GUI：
```
python main.py
```
